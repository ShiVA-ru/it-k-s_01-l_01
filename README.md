**Структура проекта**
- src — основной код приложения.
- core — здесь будут храниться общие типы, например, для HTTP-статусов.
- db — место для хранения моковых данных (в нашем случае это будет массив водителей).
- drivers — папка с кодом, связанным с водителями. Здесь будут храниться DTO (Data Transfer Object) для валидации данных и типы, описывающие водителей.
